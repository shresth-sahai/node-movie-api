# node-movie-api
Simple Node.JS Movies API

[Video Walkthrough]()

## Installation
    
  ```
    $ git clone git@github.com:kostasx/node-movie-api.git
    $ cd node-movie-api
    $ npm install
    $ node app.js
  ```

You might prefer to use **nodemon** to launch your Node.JS app instead:

  ```
    $ nodemon app.js
  ```
